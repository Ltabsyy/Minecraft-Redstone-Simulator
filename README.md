# Minecraft Redstone Simulator
**Minecraft 红石模拟器**

## 操作说明
1.左键破坏，右键放置

2.右键部分元件改变状态（0.3起）

3.右键物品栏旋转主手物品

4.按数字/滚轮切物品栏

5.Ctrl+滚轮调整大小

6.按F3显示调试信息（0.3起）

## 地图文件说明
1.程序退出时默认保存世界至`mcrs-world.txt`（0.8起）

2.程序启动时默认打开`mcrs-world.txt`（0.8起）

3.可将任意名称地图文件拖动至程序图标打开（0.8起）

4.可自行修改地图文件内`size:`后的值改变世界大小（0.8起）
